Lesson 1.1: Basics of the Web

What is the web?

The web is a bunch of computers that communicate with each other.

How does the web work?

The web has four main components:

  1. You: You want to visit a web page
  2. Your computer plus a Browser: Your browser makes a request via the internet
  3. The internet: Sends an HTTP request to a server
  4. Server: Locates appropriate HTML document and sends it back to your computer where your web browser interprets the page and displays it on your computer screen

What is a web page?

A web page is a document written in a language called HTML. Web browsers read these documents, and the interpret and display them.

What is HTML?

HTML (HyperText Markup Language) is the heart of the web.

HTML is made up of:

  • Text content (What you see)
  • Markup (What the text looks like)
  • References to other documents (eg: images, videos, etc...)
  • Links to other pages

Basic HTML vocabulary you should know:

Element: A structure in HTML that consists of an opening tag, content, and a closing tag.
Tags: Tags mark the beginning and end of an Element. They are made of a less-than
symbol, followed by the Element's name, followed by a greater-than symbol.
Attributes and values: Attributes follow tags and point to a value. Examples are
images and links to other web pages

Elements can be either inline (effecting just the text enclosed by tags), or block
(block elements have an invisible box around them with height and width that separates
them from any other element).
While most Elements have both an opening and closing tag, some Elements use only one tag
and are referred to as void.

A Couple Other Points to Remember:

Computers are stupid

Why? Because they interpret instruction literally. If you use the wrong Element
name in your opening tag or make a typo, a computer cannot "know" what you intended.

Unit 1.1: Creating a Structured Document

Part 1: How Do Websites Work?

Vocabulary:

HTML: A Language with syntax and rules that provides the structure of the web page

CSS: Provides the styles seen on the web page

Javascript: Allows actions/changes on the web page

DOM (Document Object model): The standard convention for interacting with HTML

Dig deep into code (in Chrome):

  1. Click on the symbol in the top right corner made of three lines (looks like a hamburger)
  2. Choose "More Tools" from the menu
  3. Choose "Developer Tools." A panel will open showing the structure of the web page you are on. This is not the code, but how the browser interprets it.
  4. Click on the triangles to reveal hidden levels
  5. This is the Tree-like structure of HTML

For two good articles about Tree-like Structure, check out the following Wikipedia pages:

  • Tree Structure
  • Tree (Data Structure)

Once you know the language, you can look at any HTML code and understand a great deal about how it will look as a web page. Likewise, you can view any web page and know what the underlying code looks like.

Everything in HTML is a box:

  • We organize code by determining the placement and size of the largest "boxes" first, and defining smaller and smaller boxes into those spaces.
  • Every "box" is defined by the tag "div."
  • "Divs" can be nestled endlessly within other "Divs."

HTML works with CSS:

Check out the following line:

  • <h1 class="article">

In this example, the word "class" is an attribute. It allows the user to add CSS style choices to every h1 element with the value "article."
